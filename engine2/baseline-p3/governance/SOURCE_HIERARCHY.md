# Source Hierarchy

1. Current AMC publications
Use for current exam mechanics, station count, station timing, candidate instructions, and rules.

2. Uploaded AMC Handbook
Use for station logic, MCAT structure, assessment domains, key issues, critical errors, examiner/patient performance logic.

3. Current Australian clinical references
Use for clinical safety, medication, escalation, transfer, oxygen, investigations, and do-not-delay rules.

4. Murtagh
Use for GP diagnostic framing, probability diagnosis, serious disorders not to miss, pitfalls, masquerades, and hidden agenda.
Do not use as final current medication authority when newer Australian guidance differs.

5. Uploaded feedback
Use for rhythm, traps, prompt timing, role-player behaviour, mixed recipients, and real candidate failure modes.
Do not use as clinical safety authority.
Do not copy wording.
